Linux一些我不太了解的东西

发表于 2019-01-17 | 分类于 Linux |

常用操作

sync

为了加快对磁盘文件的读写速度，位于内存中的文件数据不会立即同步到磁盘上，因此关机之前需要先进行 sync 同步操作。

磁盘

磁盘接口:IDE(ATA)(逐渐被取代),SATA,SCSI,SAS
Linux中每一个硬件都被当成一个文件,磁盘以磁盘接口类型进行命名:

  1. IDE 磁盘：/dev/hd[a-d]
  2. SATA/SCSI/SAS 磁盘：/dev/sd[a-p]

文件

文件属性

常见的文件类型及其含义有：

  1. d：目录
  2. -: 文件
  3. l: 链接文件

r、w、x 权限，表示可读、可写、可执行。

文件操作

cp

复制文件。

如果源文件有两个以上，则目的文件一定要是目录才行。

mv

移动文件

文件权限

可以将一组权限用数字来表示，此时一组权限的 3 个位当做二进制数字的位，从左到右每个位的权值为 4、2、1，即每个权限对应的数字权值为 r : 4、w : 2、x : 1。

  • 文件默认权限：文件默认没有可执行权限，因此为 666，也就是 -rw-rw-rw-.
  • 目录默认权限: 目录默认要可以进入,所以权限是777, drwxrwxrwx.

可以通过 umask 设置或者查看文件的默认权限，通常以掩码的形式来表示，例如 002 表示其它用户的权限去除了一个 2 的权限，也就是写权限，因此建立新文件时默认的权限为 -rw-rw-r–.

获取文件内容

tac

是 cat 的反向操作，从最后一行开始打印。

more

和 cat 不同的是它可以一页一页查看文件内容，比较适合大文件的查看。

less

和 more 类似，但是多了一个向前翻页的功能。

head

获得文件的前几行

指令与文件搜索

which

指令搜索,指出相应指令的文件所在地

locate

文件搜索,可以用关键字或者正则表达式进行搜索。
locate 使用 /var/lib/mlocate/ 这个数据库来进行搜索，它存储在内存中，并且每天更新一次，所以无法用 locate 搜索新建的文件。可以使用 updatedb 来立即更新数据库。

压缩与解压

gzip

tar

tar 不仅可以用于打包，也可以使用 gip、bzip2、xz 将打包文件进行压缩.

进程管理(我一直很迷…)

ps

查看某个时间点的进程信息

pstree

查看进程树,以树的形式把进程更直观的表现出来

top

实时显示进程信息

netstat

显示占用端口的进程
